动态规划介绍

动态规划通常用来求解最优化问题，而且一般是求解问题的一个最优解，能用动态规划的问题都会具有如下两个特性：

  • 最优子结构：问题的最优解由相关子问题的最优解组合而成，而这些子问题可以独立求解。

  • 子问题重叠：求解的子问题有很多重叠的步骤，可以通过存储中间过程减少求解复杂度。

求解步骤：

  • 刻画一个最优解的结构特征

  • 递归地定义最优解的值

  • 计算最优解的值，通常采用最低向上的方法

  • 利用计算出的信息构造一个最优解

动态规划一般采取自底向上解法，如上所说，通过从最底层的子问题一步步向上求解，并且记录中间问题的结果，最终得到问题的解。概念就介绍到这里，接下来看一些leetcode的真题。
